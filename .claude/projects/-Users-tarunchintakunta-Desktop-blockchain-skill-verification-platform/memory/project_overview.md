SkillChain is a blockchain-based skill verification platform with AI-powered assessment and ML job matching. It's an MSc Cloud Computing Research Project at National College of Ireland by Abhijith Kandanchira (Student ID: 23379898), supervised by Vikas Sahni. Submission due: 2026-04-23.

**Why:** Addresses a research gap — no existing system combines blockchain credentials, NFT tokenisation (ERC-721), AI assessment, ML job matching, and an integrated end-to-end platform. Closest prior work is E2C-Chain (Liu et al., 2020).

**How to apply:** All implementation work should align with the research proposal's 7 phases and the Design Science Research (DSR) methodology. The evaluation strategy (functional, assessment accuracy, matching quality) is still pending.
